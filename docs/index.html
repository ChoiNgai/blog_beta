技术

扎实的计算机基础以及开发能力是一名合格工程师的必要条件，包括但不限于数据结构、Java或Python等编程语言、SQL、大数据框架、机器学习等能力

业务

业务经验需要依托常年的阅读以及实践，优秀的数据工程师应该具备产品思维以及数据分析能力，这是以数据驱动业务的基础条件，时刻保持在业务方面的学习

视野
