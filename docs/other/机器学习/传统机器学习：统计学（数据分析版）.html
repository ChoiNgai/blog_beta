# t分布

（t分布的分子：标准正态分布；分母：卡方分布除以n开根号）

t分布用于检验均值是否不同（经常应用在对呈正态分布的总体的均值进行估计），只能检验连续型数据。

t统计量的平方等于FFF统计量，大样本时，ttt检验等价于ZZZ检验，其平方等效于卡方统计量。

由于在实际工作中，标准差σσσ往往是未知的，常用s作为σ的估计值，为了与μ\muμ变换区别，因而称为ttt变换，统计量ttt 值的分布称为ttt分布。

（备注：μ\muμ变换将X∼N(μ,σ2)X \sim N( \mu,\sigma^2)X∼N(μ,σ2)变换成X∼N(0,1)X \sim N(0,1)X∼N(0,1)，变换公式为：X−μσ\frac{X-\mu}{\sigma}σX−μ​，由正态分布的性质可知变换后服从标准正态分布）

定义

假设X∼N(0,1),Y∼χ2(n)X\sim N(0,1),Y \sim \chi^2(n)X∼N(0,1),Y∼χ2(n)，则Z=X(Y/n)Z=\frac{X}{\sqrt{(Y/n)}}Z=(Y/n)​X​的分布称为自由度为nnn的ttt分布
