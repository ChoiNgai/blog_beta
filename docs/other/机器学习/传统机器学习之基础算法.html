# 全监督分类算法：KNN

与kmeans经常提起的是一种分类方法：K最近邻分类（KNN），这是一种较为简单的分类算法（有监督，分类算法通常都是指有监督的，虽然也有无监督的分类算法）

具体过程，KNN没有迭代，给定样本是已有标签（类别），输入新的样本（即需要预测的样本），计算新的样本与各个点之间的距离，将距离排序，选取k个距离最小值，看k个距离最小值对应的样本点是哪一类，根据少数服从多数原则判断出新的样本属于哪一类，结束。

# 模糊聚类开山作：FCM

模糊聚类又称为软聚类，相对于硬聚类这种非此即彼的分类，其用隶属度来表示每个样本属于每一类的程度。

模糊聚类中最为经典(古老)的是模糊C均值聚类（FCM），但在里哦了解FCM之前需要先掌握kmeans。在kmeans中，聚类中心$ v_i $确定之后就确定样本所属类别了，属于其他类别的程度为 0。在模糊聚类中，增加了一个隶属度来表示各个样本属于每一类的程度。

在模糊聚类中通常用uiju_{ij}uij​来表示隶属度矩阵，其中i={1,..c}i=\{1,..c \}i={1,..c}，j={1,2,...n}j= \{ 1,2,...n \}j={1,2,...n}；ccc为类簇数，nnn为样本数。（跟上面介绍kmeans的符号是一样的）

模糊C均值聚类的目标函数：

J(U,V)=∑i=1c∑j=1nuijm∣∣xj−vi∣∣2J(U,V) = \sum_{i=1}^c \sum_{j=1}^n u_{ij}^m ||x_j-v_i||^2 J(U,V)=i=1∑c​j=1∑n​uijm​∣∣xj​−vi​∣∣2

s.t.uij∈[0,1];∑i=1cuij=1s.t. \quad u_{ij} \in [0,1]; \sum_{i=1}^c u_{ij}=1 s.t.uij​∈[0,1];i=1∑c​uij​=1

其中ccc是类簇数（C均值聚类嘛，所以相比kmeans把k换成了c）。mmm是加权指数幂，是一个自己给定的大于2的参数。

目标函数的约束条件是隶属度的范围是0~1，每个样本的隶属度之和为1。

上述目标函数只有$ u_{ij} 和和和v_i $是未知，其他都为已知。

  • **随机初始聚类中心/隶属度矩阵:**从上面的隶属度矩阵和聚类中心迭代式可看出，隶属度矩阵迭代式里的未知只有viv_ivi​，而聚类中心跌打式里不知道的只有uiju_{ij}uij​。故迭代前先随机给定一个符合条件的隶属度矩阵或聚类中心矩阵，然后即可开始迭代。

# 降维

# 主成分分析(PCA)

主成分分析是最常用的一种降维方法，通过正交变换将一组可能存在相关性的变量转换成一组线性不想关的变量，转换后的变量称为主成分。

（主成分分析是一种特征提取方法，提取后的特征不是原数据的子集）

PCA 实质上是一个基变换，使得变换后的数据有**最大的方差（因为统计学里某个变量方法越大，说明这个变量越有特征显著性），也就是通过对坐标轴的旋转和坐标原点的平移使得其中一个轴（主轴）与数据点之间的方差最小，坐标转换后去掉高方差的正交轴，得到降维数据集。

主成分分析可采用特征分解或奇异值分解（SVD），SVD分解具体如下：

X=U∑VTX=U \sum V^TX=U∑VT

其中XXX是原始数据，UUU是正交矩阵，

Y=U∑Y = U \sumY=U∑

参考：http://c.biancheng.net/view/1951.html

具体步骤：

  1. 每个变量（特征）减去其对应的均值
  2. 求特征协方差
  3. 求协方差的特征值和特征向量
  4. 特征值从大到小排序，选择最大的k个；将k个特征向量作为列向量组成特征向量矩阵（投影矩阵）
  5. 原数据与特征向量矩阵相乘，得到降维数据

主成分分析是直接对数据进行线性变换，这种方法在降维上起到了良好的效果，但是对于某些场景（暂时不清楚什么场景），流形学习降维或许有更好的效果（区别：流形学习保留了数据的局部结构）；对于有监督的场景，特征选择或许比主成分分析更合适
