# 统计学

# 第一章、导论

统计方法

  • 描述统计：
    • 是研究数据收集、处理、汇总、图表描述、概括与分析等统计方法
    • 目的：描述数据特征，找出数据的基本规律
    • 包含：
      • 集中趋势分析（平均数，中位数，众数）
      • 离中趋势分析（全距，四分差，平均差，方差，标准差）
      • 相关分析（探讨数据与数据之间是否有关联性）
  • 推断统计：
    • 是研究如何利用样本数据来推断总体特征的统计方法
    • 目的：对总体特征做出判断
    • 包含：
      • 参数估计
      • 假设检验

统计数据分类

  • 分类数据
    • 是按照现象的某种属性对其进行分类或分组而得到的数据
    • 分类数据之间没有数量上的关系和差异。如，用1表示“男性”，0表示“女性”，但是1和0等只是数据的代码
  • 顺序数据
    • 是只能归于某一有序类别的非数字型数据
    • 顺序数据：顺序数据之间是有序的。如表示受教育程度可以分为小学、初中、高中、大学及以上
  • 数值型数据
    • 是按数字尺度测量的观察值，其结果表现为具体的数值
  • 观测数据
    • 又称原始数据，是指在自然的未被控制的条件下观测到的数据
  • 实验数据
    • 是指在实验中控制实验对象而搜集到的变量的数据
  • 截面数据
    • 也称静态数据，是不同主体在同一时间点或同一时间段的数据
  • 时间序列数据
    • 是在不同时间上收集到的数据，用于所描述现象随时间变化的情况。这类数据反映了某一事物、现象等随时间的变化状态或程度

几个基本概念

  • 总体

    • 所研究的全部个体（数据）的集合，其中的每一个个体也称为元素
    • 分为有限总体和无限总体

  • 样本

    • 从总体中抽取的一部分元素的集合

  • 参数

    • 用来描述总体特征的概括性数字度量
    • 通常用希腊字母表示

  • 估计量

    • 用来估计总体未知参数用的统计量

  • 样本量

    • 构成样本的元素的数目称为样本容量或样本量（只是一个数字）

  • 统计量

    • 用来描述样本特征的概括性数字度量
    • 通常用小写英文字母表示

  • 变量

    • 说明现象某种特征的概念
    • 分类：
      • 分类变量
        • 说明事物类别的一个名称，其取值是分类数据
      • 顺序变量
        • 其值仅表明事物属性在数量大小、多少上的次序的变量
      • 数值型变量
        • 离散型变量：取有限个值
        • 连续型变量：可以取无穷多个值
      • 随机变量
        • 表示随机试验各种结果的实值单值函数。随机事件不论与数量是否直接有关，都可以数量化，即都能用数量化的方式表达
      • 非随机变量
      • 经验变量（周围可以观察到的事物）
      • 理论变量（用数学方法构造出的变量，z统计量，t统计量）

# 第二章、数据的收集

数据的来源

  • 数据的间接来源
  • 数据的直接来源

调查方法

  • 概率抽样
    • 简单随机抽样
      • 设一个总体含有N个个体，如果通过逐个抽取的方法从中抽取一个样本，且每次抽取时各个个体被抽到的概率相等
      • 是其他抽样方法的基础
    • 分层抽样
      • 将抽样单位按某种特征或某种规则划分为不同的层，然后从不同的层中独立、随机地抽取样本
    • 整群抽样
      • 将总体中若干个单位合并为组，这样的组称为群。抽样时直接抽取群，然后对中选群中的所有单位全部实施调查
    • 系统抽样
      • 又叫等距抽样，将总体中的所有单位按照一定顺序排列，在规定的范围内随机抽取一个单位作为初始单位，然后按实现定好的规则确定其他样本单位
    • 多阶段抽样
      • 首先抽取群，但并不是调查群内的所有单位，而是再进一步抽样，从选中的群中抽取出若干个单位进行调查
    • 优点：可以依据调查结果计算估计量误差
  • 非概率抽样
    • 方便抽样，判断抽样，自愿抽样，滚雪球抽样，配额抽样

搜集数据的基本方法

  • 自填式，面访式，电话式

实验方法

数据的误差

  • 抽样误差
    • 是由抽样的随机性引起的样本结果与总体真值之间的差异
    • 是指由于随机抽样的偶然因素使样本各单位的结构不足以代表总体各单位的结构，而引起抽样指标和全局指标的绝对离差
    • 样本量越大，抽样误差越小
    • 只存在于概率分布
  • 非抽样误差
    • 是指除抽样误差之外的，由其他原因引起的样本观察结果与总体真值之间的差异
    • 是一种随机性误差，无论什么抽样中都会有
    • 包括：抽样框误差，回答误差，无回答误差，调查员误差，测量误差
  • 误差的控制

数据的预处理

  • 数据排序（按一定顺序将数据排列）

品质数据的整理与展示

  • 分类数据的整理与图示
    • 频数与频数分布
      • 落在某个特定类别或组中的数据个数
    • 比例，百分比，比率（比例之间的比值）
    • 分类数据的图示

      • 帕累托图（左边是计数，右边是累计百分比）
      • 饼图（百分比）
      • 环形图（饼图是针对一维变量，环形图针对二维变量，如：饼图是研究不同类型饮料的构成，环形图是研究按性别绘制的不同类型饮料的构成）
  • 顺序数据的整理与图示
    • 累积频数和累积频率
      • 是将各有序类别的频数逐级累加起来得到的频数
      • 是将各有序类别或组的百分比逐级累加
    • 顺序数据的图示
      • 累计频数分布或频率图

数值型数据的整理与展示

  • 数据分组

    • 数据分组，组距分组，等距分组，不等距分组，组距，组中值

  • 数值型数据的图示

    • 直方图（横轴是分组，竖轴是频数）

    • 茎叶图，箱线图，条形图，线图，雷达图

# 第四章、数据的概括性度量

集中趋势的度量

  • 众数（没有，一个，多个）
  • 中位数
    • 个数为奇数，取中间的那个
    • 个数为偶数，中间两个数的平均值
  • 四分位数
    • 排序后处于25%和75%位置上的值
    • 25%
      • 为整数，直接取
      • 为分数，左右两个数取平均值
    • 75%（道理同上）
  • 平均数（分为样本和总体）
    • 简单平均数
    • 加权平均数
    • 几何平均数
      • n个变量值乘积的 n 次方根

离散程度的度量

  • 异众比率（分类数据）
    • 对分类数据离散程度的测度，非众数组的频数占总频数的比例
  • 四分位差（顺序数据）
    • 上四分位数与下四分位数之差（75%-25%）
    • 反映了中间50%数据的离散程度
  • 极差
    • 最大值减最小值
  • 平均差
    • 各变量值与其平均数的差的绝对值的平均数
  • 方差（无开方）和标准差（有开方）（数值型数据）
  • 自由度
    • 是指一组数据中可以自由取值的个数
    • 注意：样本方差的自由度为n-1
  • 标准分数（相对位置的度量）
    • 对某一个值在一组数据中相对位置的度量
    • 某一点的值减去平均值除以标准差
  • 经验法则
    • 前提：对称分布
    • 约有68%的数据在平均数加减1个标准差的范围之内
    • 约有95%的数据在平均数加减2个标准差的范围之内
    • 约有99%的数据在平均数加减3个标准差的范围之内
    • 3倍标准差以外的数据称为离群点
  • 切比雪夫不等式
    • 指在任何数据集中，与平均数超过k倍标准差的数据占的比例至多是1/k的平方（k不一定是整数）
    • 例如：
      • 对于k=2，3，4，该不等式的含义
      • 至少有75%的数据落在平均数加减2个标准差的范围之内
      • 至少有89%的数据落在平均数加减3个标准差的范围之内
      • 至少有94%的数据落在平均数加减4个标准差的范围之内
  • 离散系数（相对离散程度）
    • 又称变异系数，是标准差与其相应的均值之比
    • 是对数据相对离散程度的测度
    • 主要用于比较不同样本数据的离散程度

偏态与峰态的度量

  • 偏态
    • 数据分布偏斜程度的测度（数据侧重于哪边）
    • 偏态系数
      • 偏态系数=0为对称分布
      • 偏态系数> 0为右偏分布（又叫正偏态）（数据在左半边突起，在右半边凹陷，即平均数在众数的右边）
      • 偏态系数< 0为左偏分布（又叫负偏态）（数据在右半边突起，在左半边凹陷，即平均数在众数的左边）
    • 偏态分布
      • 偏态系数大于1或小于-1，是高度偏态分布
      • 偏态系数在0.5～1或-1～-0.5之间，是中等偏态分布
      • 偏态系数越接近0，偏斜程度就越低
  • 峰态
    • 数据分布扁平程度的测度（数据集中处的集中程度，尖度）
    • 峰态系数
      • 峰态系数=0扁平峰度适中
      • 峰态系数> 0为扁平分布
      • 峰态系数< 0为尖峰分布

# 第五章、概率与概率分布

随机事件及其概率

  • 事件
    • 随机事件
      • 在同一组条件下，每次实验可能出现也可能不出现的事件
    • 必然事件
      • 在同一组条件下，每次实验一定出现的事件
    • 不可能事件
      • 在同一组条件下，每次实验一定不会出现的事件
    • 基本事件
      • 一个事件不能分解成两个或更多个事件
  • 概率
    • 概率的古典定义
    • 概率的统计定义
    • 主观概率定义

概率的性质与运算法则

  • 主要就是概率论的内容

离散型随机变量及其分布

  • 如果随机变量X只可能取有限个或至多可列个值，则称X为离散型随机变量

    • 二项分布，泊松分布
  • 性质：
    • 二项分布的数学期望为np，方差为np（1-p）

连续性随机变量及其分布

  • 是指如果随机变量X的所有可能取值不可以逐个列举出来，而是取数轴上某一区间内的任一点的随机变量
  • 性质：
    • 概率密度函数
      • 大小大于零，面积等于一
      • 随机变量取值在a与b之间的概率写为a与b之间的面积
      • 数学期望：
      • 方差：
    • 概率分布函数
      • 其导数为概率密度函数
    • 正态分布

# 第六章、统计量及其抽样分布

统计量

  • 设X1,X2,…,Xn是从总体X中抽取的容量为n的一个样本，如果由此样本构造一个函数T(X1,X2,…,X**n)，不依赖于任何未知参数，则称函数T(X1,X2,…,Xn)是一个统计量

  • 是样本的一个函数，是统计推断的基础

  • 如：样本均值，样本比例，样本方差，样本变异系数，样本的k阶矩，k阶中心矩，样本偏度

  • 次(顺)序统计量

    • 设 X1,X2, …, Xn是取自总体X的样本，X(i) 称为该样本的第i个次序统计量，它的取值是将样本观测值由小到大排列后得到的第i个观测值。从小到大排序为x(1),x(2), …,x(n)，则称X(1),X(2), …,X(n)为顺序统计量
    • 如：四分位极差，极差，中位数，最大值，最小值

  • 充分统计量

    • 对于给定的统计推断问题，包含了原样本中关于该问题的全部有用信息的统计量。

      对于未知参数的估计问题，保留了原始样本中关于未知参数θ的全部信息的统计量，就是充分统计量。

      如样本均值X是总体数学期望的充分统计量。数学上，设(X₁， …，Xₑ)是来自总体X的一个随机样本，T=T(X₁， …，Xₑ)是一统计量。若在T=t的条件下，样本的条件分布与未知参数θ无关，则称统计量T是θ的充分统计量

    • 解释：简单的说，知道了充分统计量我们就可以扔掉样本，因为样本里的信息全都被包含在充分统计量里面。

    • 经证明，次序统计量就是充分统计量（次序的范围小于充分）

关于分布的几个概念

  • 抽样分布
    • 也称统计量分布、随机变量函数分布，是指样本估计量的分布
    • 是统计推断的理论基础
    • 抽样分布、参数估计和假设检验是统计推断的三个中心内容
  • 渐近分布
  • 随机模拟获得的近似分布

由正态分布导出的几个重要分布

# 第七章、参数估计

参数估计的基本原理

  • 估计量
    • 用来估计总体未知参数用的统计量

    • 在数理统计中，一般用子样观测值求出的统计量来估计总体的一个未知参数，此统计量称为参数的估计量。子样一组观测值所对应估计量的值，称为参数的估计值

# 第八章、假设检验

假设检验的基本问题

  • 问题的提出

    • 假设检验

      • 假设检验的过程，可以用法庭的审理来说明。先想像现在法庭上有一名被告，假设该被告是清白的，而检察官必须要提出足够的证据去证明被告的确有罪。 在证明被告有罪前，被告是被假设为清白的。

        • 假设被告清白的假设，就相当于[零假设]（null hypothesis）。
        • 假设被告有罪的假设，则是[备择假设]（alternative hypothesis）。

        而检察官提出的证据，是否足以确定该被告有罪，则要经过检验。 这样子的检验过程就相当于用T检验或Z检验去检视研究者所搜集到的统计资料。

      • 零假设：欲检验其正确性

      • 备择假设：而站在零假设立场对立面的即为备择假设

  • 两类错误

    • 第一型错误
      • 若零假设事实上成立，但统计检验的结果拒绝零假设（接受备择假设）（发生概率为α）
    • 第二型错误
      • 若零假设事实上不成立，但统计检验的结果不拒绝零假设（接受零假设）（发生概率1-β）

  • 假设检验的流程

    • 最初研究假设为真相不明。
    • 提出相关的零假设和备择假设。
    • 考虑检验中对样本做出的统计假设；例如，关于总体资料的分布形式或关于独立性的假设。无效的假设将意味此检验的结果是无效的。
    • 选择一个显著性水平 (α)，若低于这个概率阈值，就会拒绝零假设。最常用的是 5% 和 1%。
    • 选择适合的检验统计量 T。
    • 在设定零假设为真下推导检验统计量的分布。在标准情况下应该会得出一个熟知的结果。比如检验统计量可能会符合正态分布或司徒顿t分布。
    • 根据在零假设成立时的检验统计量T分布，找到概率为显著性水平 (α)的区域，此区域称为“拒绝域”，即在零假设成立的前提下，落在拒绝域的概率只有α。
    • 针对检验统计量T，根据样本计算其估计值tobs。
    • 若估计值tobs未落在拒绝域，则“不拒绝”零假设（do no reject {\displaystyle H_{0}}H_0）。若估计值tobs落在拒绝域，则拒绝零假设，接受备择假设

  • 利用p值进行决策

列联表分析：独立性检验

  • 列联表
    • 是将两个以上的变量进行交叉分类的频数分布表
  • 独立性检验
    • 分析列联表中行变量和列变量是否相互独立
    • 方法：与拟合优度检验同理（注意：这里是总体要乘以两个概率，行和列的）

列联表中的相关测量

  • φ 相关系数

    • 是描述2*2的列联表数据相关程度最常用的一种相关系数

    • $$ φ=\sqrt{χ²/n} $$

    • 取值范围在0-1之间，绝对值越大，代表两个变量的相关程度越高

    • n是总频数

    • 但是党行数或是列数大于2时，φ系数将会变大，没有上限，所以需要列联相关系数

  • 列联相关系数（又称c系数）

    • 主要用于列联大于2*2的情况

    • $$ c=\sqrt{χ²/(χ²+n)} $$

    • 相互独立的时候为0，且不可能大于1

    • 鉴于φ 相关系数无上限，c系数又小于1，所以有了V系数

  • V相关系数

    • $$ V=\sqrt{χ²/n*min[(R-1),(C-1)]} $$

    • 使用范围都可以

    • min[(R-1),(C-1)]是取其中最小的一个

    • 当二者相互独立的时候，为0；完全相关的时候为1；值在0-1之间

列联分析中应该注意的问题

  • 条件百分表的方向
    • 如果变量X与Y存在因果关系，令X为自变量，Y为因变量，那么一般做法是把自变量X放在列的位置
  • χ²分布的期望值准则
    • 如果只有两个单元，则每个单元的期望频数必须是5或5以上
    • 倘若有两个以上的单元，20%的单元的期望频数小于5，则不能用χ²检验

# 第十章、方差分析

单因素方差分析

  • 方差分析中只有一个分类型变量

  • 方法步骤：

    • 提出假设
    • 构造检验的统计量
      • 计算各样本的均值
      • 计算全部观测值的总均值
      • 计算各误差平方和
        • 总平方和（SST）
          • 是全部观测值与总均值的误差平方和
          • 就每个观测值减去总均值的平方再相加
          • 反映了自变量和残差变量的共同影响
        • 组间平方和（SSA）
          • 是各组均值与总均值的误差平方和，
          • 就每组的均值减去总均值再乘以每组的个数再相加
          • 是对随机误差和系统误差的度量，反映自变量对因变量的影响，也称自变量效应或因子效应
        • 组内平方和（SSE）
          • 是每个水平或组的各样本数据与其均值的误差平方和，又称残差变量，又称误差平方和
          • 就每个观测值减去每组的均值再相加
          • 是对随机误差的度量，反映每个样本各观测值的离散情况，也称残差效应
        • 关系：SST=SSA+SSE
      • 计算统计量
        • SST的自由度为n-1，n为全部观测值的个数
        • SSA的自由度为k-1，k为因素水平（总体）的个数
        • SSE的自由度为n-k
        • SSA的均方MSA=组间平方和/自由度=SSA/n-1
        • SSE的均方MSE=组内平方和/自由度=SSE/N-k
        • 将MSA和MSE进行对比，得到所要检验的统计量F，F=MSA/MSE
    • 做出统计决策
      • 找到显著性水平和两个对应的自由度，查表
