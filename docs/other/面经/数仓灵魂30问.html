# 数仓灵魂30问

# 0.自我介绍

面试官好,我叫蔡炜，广东工业大学应用统计学22届学生，很荣幸能够参加字节的数据仓库实习生面试。

我从18年6月份开始在编程方面的学习，至今手写代码超过10w行，在前端、后端、客户端等均有所了解。在大学期间参与过大大小小的比赛十余次，发表/投稿论文10篇，申请专利软件著作权等10余项。

今年3月份决定工作，4月中旬入职欢聚时代，岗位是数据开发实习生，所在小组负责电商领域的数据仓库。

虽然我在大数据开发-数据仓库方面还有很多知识亟须补充，但已明确自己的职业发展方向是在数据仓库领域，我的介绍到此结束。

  • 维度建模 星座模型 宽表 粒度 指标体系
  • 任务调度 数据质量 元数据管理 血缘关系 数据治理
  • BI可视化 OLAP多维分析 用户画像 推荐系统
