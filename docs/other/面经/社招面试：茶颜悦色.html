# 茶颜悦色

笔试题

一，数仓部分

1.商品活动表goods_activity,字段id主键，goods_id 商品id，act_id 活动id，create_time

活动创建时间，status 上线状态（2 上线，3 下线）

问题一，每个活动的每个商品的开始时间和结束时间？

问题二，假设数据中有脏数据，同一个活动可能上线两次，也可能结束两次，需要把活动上线重复的状态按照时间取第一条，结束上线状态重复的按时间取最后最后一条

2.请简述以下数据仓库一般有几种设计模式，以及每种设计模式的优缺点？

3.假设现在有一个社保缴费清单表A（社保号，参保单位编号，缴费月份），缴费清单保存个人历史所有参数记录，请用一个SQL（可伪代码）统计出每个参保人（以社保标识一个人）在每段工作经历（以缴纳社保为准）的最早缴社保月份，最近缴社保月份，缴纳月份数）。

4.数据仓库如何分层，为什么？

5.介绍以下数据倾斜，简述产生的原因及解决方案都有哪些？

二，大数据开发部分

1.列举一下用过的大数据工具，简述每种工具的用途以及优缺点？

2.简述Hbase filter的实现原理是什么？结合实际项目经验，写出一个filter场景？

3，Flume和Kafka采集日志区别，采集日志停了中间停了，怎么记录之前的日志？

4，Sqoop在导入数据到mysql中，如何让数据不重复导入？在Sqoop如何处理？

5，请列出你所知道的hadoop调度器，并简要说明其方法？
